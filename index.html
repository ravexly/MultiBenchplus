About the Project

Unified Benchmark

Standardized evaluation across diverse multimodal tasks including emotion recognition, healthcare, and remote sensing.

Extensive Datasets

Coverage of 39 datasets across various domains, ensuring robust and comprehensive model assessment.

Easy Integration

Modular codebase designed for easy extension, allowing researchers to plug in new models and datasets effortlessly.

Advanced Metrics

Detailed performance analysis using state-of-the-art evaluation metrics tailored for each specific domain.

MultiBench++ implements a comprehensive taxonomy of multimodal fusion approaches. The following tree structure organizes fusion methods by their integration stage and architectural characteristics.

Multimodal Fusion Methods
  • Early Fusion — Feature-level integration
    • Concatenation
      Concat ConcatEarly
    • Tensor Fusion
      TensorFusion LowRankTensorFusion
    • Multiplicative Interactions
      MultiplicativeInteractions2Modal MultiplicativeInteractions3Modal
  • Intermediate Fusion — Joint representation learning
    • Transformer-Based
      EarlyFusionTransformer LateFusionTransformer

    • Non-Local Gating

Dataset Loaders

dataset/*/get_data.py

Unified data loading interface for all 39 datasets. Each dataset has its own loader with consistent API, making it easy to switch between datasets.

MAMI MELD IEMOCAP MIMIC-III SIIM-ISIC Houston2013 ForestNet SUN-RGBD +31 more
02

Unimodal Encoders

encoders/

Pre-built encoders for extracting features from different modalities. Supports both pre-trained and custom architectures.

BERT Encoder bert.py

Pre-trained BERT-base-uncased for text encoding. Supports fine-tuning and feature extraction modes.

Image Encoder image.py

ResNet, VGG, and custom CNN architectures for visual feature extraction.

03

Training & Evaluation

training_structures/ & eval_scripts/

Flexible training pipelines and comprehensive evaluation metrics for benchmarking multimodal models.

Training Structures

Supervised_Learning_V2.py provides standard supervised training loop with support for gradient blending, multi-task learning, automatic checkpointing and early stopping.

Performance Metrics

performance.py includes accuracy, F1, AUC, and domain-specific metrics like C-index for survival and mIoU for segmentation.

Robustness Testing

robustness.py evaluates missing modality handling, noise injection, adversarial testing, and cross-dataset generalization.

Complexity Analysis

complexity.py measures FLOPs, parameter counting, memory profiling, inference speed, and efficiency benchmarking.

Structure & Usage

Project Hierarchy

MultiBench++/
├── datasets/             # Loaders for 10+ domains (MAMI, etc.)
├── encoders/             # Unimodal backbones (BERT, ResNet)
├── fusions/              # 15+ fusion implementations
│   ├── early_fusion.py   # Concatenation & Tensor Fusion
│   ├── attention.py      # Cross-modal Attention
│   └── late_fusion.py    # Evidence Theory & Decision fusion
├── training_structures/  # Pipelines (GradientBlend, MMD)
├── objective_functions/  # Loss functions & metrics
├── configs/              # Hyperparameter configs
├── main.py               # Main experiment entry point
└── requirements.txt      # Dependencies

Quick Start

# 1. Environment Setup
conda create -n multibench python=3.9
conda activate multibench
pip install -r requirements.txt
# 2. Data Preparation
# Download or place data in ./data
bash scripts/download_data.sh
# 3. Run Experiments
# Example: Train Tensor Fusion on MAMI
python main.py --dataset MAMI --fusion tensor_fusion
# 4. Evaluation
python eval.py --ckpt checkpoints/best.pth
